3. Data Collected

The script may process:

  • Job-related information entered manually into the worksheet
  • Staff information from the Staffs_WorkID sheet
  • User inputs required for managing data (e.g., job entries, staff hours)

Note: All data remains within the Google Sheet environment and is never transmitted, shared, or stored outside the user's Google Drive.

4. Purpose of Data Processing

The script processes data solely for the following purposes:

  • Automating workflows within the worksheet
  • Ensuring data consistency across multiple sheets
  • Enhancing user experience by reducing manual operations

No personal data is collected, stored, or transmitted outside the worksheet environment.

5. Access and Sharing

The script operates with permissions granted through Google Apps Script. Only users with edit access to the worksheet can execute the script. No data is shared with third parties or external services.

6. Data Security

The script runs exclusively within Google Sheets, ensuring compliance with Google's security standards. Access to the script is restricted to authorized users of the worksheet.

7. User Rights

As a user, you have the right to:

  • Modify or delete data within the worksheet
  • Stop using the script at any time by removing it from the Apps Script editor

This Privacy Policy may be updated to reflect changes in the functionality or scope of the script. Users will be notified of any updates through the worksheet.
